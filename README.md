# Answer-me

根据指定命令回复相应内容。

---

## 📘 使用教程

建议使用配置文件方式管理回复规则。

### 📄 配置文件说明

配置文件位于项目根目录下的 `data/answer-me.json`，用于定义命令与回复内容的映射关系。

---

### 回复类型

#### 1. 单文字回复

```json
{
    "command": "我是命令词",
    "text": "我是回复词"
}
```

#### 2. 单图片回复

```json
{
    "command": "我是命令词",
    "image": "./****/****.jpg"
}
```

#### 3. 图文回复

```json
{
    "command": "我是命令词",
    "replyText": "我是回复词",
    "replyImages": "./****/****.jpg"
}
```

#### 4. 文件回复

```json
{
    "command": "我是命令词",
    "replyFiles": ["./****/****.txt"]
}
```

---

### 额外参数

| 参数 | 说明 | 支持格式 |
|:-----|:-----|:---------|
| `alias` | 命令别名 | 路径数组 |
| `image` | 图片路径 | 路径数组 |
| `replyImages` | 回复图片 | 路径数组 |

> **提示**：`image`、`alias` 等参数均支持数组形式，可配置多个值。
